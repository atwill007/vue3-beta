# vue3-beta


## 关于Vue3的六大亮点
- Performance
  - 性能更比Vue 2.0强，主要体现在包的的大小和运行效率上
- Tree shaking support
  - 可以将Vue的无用模块移除，仅打包需要的功能
- Composition API
  - 组合API，给代码组织提供更多的可能性
  - [代码示例](./src/views/CompositionAPI.vue)
  - [官方文档](https://composition-api.vuejs.org/)
- Fragment, Teleport, Suspense
  - Fragment 碎片
  - Teleport 即Protal传送门
  - Suspense 悬念
- Better TypeScript support
  - 更优秀的TS支持
- Custom Renderer API
  - 暴露了自定义渲染API
